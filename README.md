# Insta-Pixy-Chat-App

# Aim:
The aim of the project is to create a  realtime Chat Application where multiple users can have chat with each other.

# Languages and Frameworks Used:
- HTML5
- CSS3
- JavaScript
- NodeJS
- ExpressJS
- Socket.io
